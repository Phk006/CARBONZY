# Project Overview

## Purpose

CARBONZY is designed as a student-friendly sustainability application. The goal is to make low-carbon habits feel visible, rewarding, and easy to repeat on campus.

## Target Users

- University students
- Campus sustainability clubs
- Student affairs teams
- Eco-conscious student communities

## Core Idea

Students earn coins by completing daily sustainability actions:

- Walking instead of using motor transport
- Carrying a reusable bottle
- Avoiding plastic bags
- Cycling to class
- Switching lights off
- Choosing plant-based meals
- Composting waste
- Verifying plastic disposal

Coins can then be redeemed in an eco marketplace.

## Main Screens

- Splash screen: app entry point
- Track: daily score, steps, habit cards, recycling shortcut
- Challenges: weekly sustainability goals
- Shop: eco product redemption catalog
- Recycle: camera-based plastic disposal flow
- Profile: badges, lifetime coins, score, and purchases

## Current Scope

This is a static front-end prototype. It is suitable for demos, presentations, hackathons, and early user testing.

## Out Of Scope For Current Version

- Real authentication
- Persistent database storage
- Real payment or checkout integration
- Real computer vision verification
- Admin dashboard
- Multi-user leaderboard persistence
