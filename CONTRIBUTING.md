# Contributing

Thanks for helping improve CARBONZY.

## Local Setup

Clone the repository:

```bash
git clone https://github.com/Phk006/CARBONZY.git
cd CARBONZY
```

Open `index.html` directly in a browser, or serve it locally:

```bash
npx serve .
```

## Development Guidelines

- Keep the app mobile-first.
- Avoid adding build steps unless they are necessary.
- Keep user-facing text simple and student-friendly.
- Test the main flows after every change:
  - Start app
  - Add steps
  - Complete habits
  - Open challenges
  - Redeem a product
  - Open profile
  - Run the recycle demo flow

## Commit Style

Use short, clear commit messages.

Examples:

```text
Improve market product cards
Add local storage persistence
Fix camera fallback preview
```

## Pull Requests

Include:

- What changed
- Why it changed
- How it was tested
- Screenshots for visual changes
